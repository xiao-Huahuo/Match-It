## 技术栈

## 设计功能体系
### 公共模块
- 社团主页展示（简介、活动、成员、公告）
- 可切换“暗黑/明亮”主题
- 首页轮播图支持“一键外链跳转”到赞助商
### 社团管理员模块
- 创建、编辑、解散社团
- 社团成员增删(支持批量导入和批量删除)
- 角色权限和席位设置
- 发布活动:
  - 社团招新报名
  - 投票
  - 意见收集
  - 发布公告
- 针对特定成员或普通用户发送消息
  - 入社申请
  - 活动提醒
  - 系统公告
  - 缴费提醒
### 普通用户模块
- 用户注册 / 登录 / 退出
- 个人信息管理（头像、简介、兴趣标签）
- 权限分级：普通用户、社团成员(With 席位)、社团管理员  (USER,MEMBER,ADMIN)
### 社员活动管理模块
- 社员可报名、签到、反馈
- 经费报销流程器
### 智能匹配与智能交互模块
- 基于兴趣标签、行为数据推荐社团各职位
- 匹配算法示例：
$$
score(user,club)=\frac{|tag\ s_{user} \cap tag\ s_{club}|}{|tag\ s_{user} \cup tag\ s_{club}|}
$$
- 后期可引入 NLP / Embedding 模型优化语义匹配
- 内置LLM智能体(**KIMI**)进行问答和内容检索
  - LLM 问答日志落库，后台可查看高频提问，反向生成“社团百科”词条，减少重复提问。
### 消息与通知模块
- 通知分级：普通（站内）、重要（邮件）、紧急（短信）
- 个人邮箱
- 可扩展为即时通信（Chat）子模块
### 数据分析模块(部分管理员可见,部分社员可见)
- 统计社团活跃度、活动参与度
- 物资借用台账
- 可视化展示数据趋势（ECharts）
- 成员考勤记录表,反馈收集
### 日志与备份模块(管理员可见)
- 操作日志每日自动导出
- 每晚定时导出SQL关键表到 .sql 文件，支持一键恢复页面
