# Match It !
##### ——社团智能联动匹配系统

> "你好,我想要面试贵社,以下是我的简历"<br>
> "什么?没打过挑战杯?不要不要,另寻别处吧."
### 技术栈
<a href="https://spring.io/projects/spring-boot">SpringBoot</a> + <a href="https://cn.vuejs.org/">Vue3</a> + <a href="https://www.mysql.com/">MySQL</a>

### 设计功能体系
### （1）用户系统模块
- 用户注册 / 登录 / 退出
- 个人信息管理（头像、简介、兴趣标签）
- 权限分级：普通用户、社团负责人、管理员
### （2）社团管理模块
- 创建、编辑、解散社团
- 社团成员增删与角色权限设置
- 社团主页展示（简介、活动、成员、公告）
### （3）活动管理模块
- 社团可发布活动
- 学生可报名、签到、反馈
- 后台自动统计参与人数与活跃度
### （4）智能匹配模块（核心）
- 基于兴趣标签、行为数据推荐社团
- 社团端可查看潜在成员列表
- 匹配算法示例：
$$
score(user,club)=\frac{|tag\ s_{user} \cap tag\ s_{club}|}{|tag\ s_{user} \cup tag\ s_{club}|}
$$

- 后期可引入 NLP / Embedding 模型优化语义匹配
### （5）消息与通知模块
- 入社申请、活动提醒、系统公告
- 支持站内信与邮件提醒
- 可扩展为即时通信（Chat）子模块
### （6）数据分析模块
- 统计社团活跃度、活动参与度
- 可视化展示数据趋势（ECharts）
- 为社团管理决策提供参考
